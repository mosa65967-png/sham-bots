# شام بوتس - Sham Bots 🇸🇾

**المنصة الرقمية السورية المتكاملة** - SaaS متعدد الخدمات يدعم 14 محافظة سورية.

## 🚀 الخدمات

- 🤖 **بوتات ذكية** - تليجرام، واتساب، فيسبوك، إنستغرام مع AI
- 📋 **دليل تليجرام** - أكبر دليل للقنوات والمجموعات والبوتات
- 🍽️ **QR Menu** - منيو رقمي تفاعلي للمطاعم
- 🛒 **متجر إلكتروني** + دروب شيبينج
- 🎧 **SaaS خدمة عملاء** مع رد آلي بالذكاء الاصطناعي

## 💳 نظام الدفع

- **نقداً باليد** (يد بيد / توصيل للمنزل)
- **قابل للتوسع** ليشمل Stripe/PayPal لاحقاً
- **عملتان**: الليرة السورية والدولار

## 🗺️ المحافظات المدعومة

دمشق، ريف دمشق، حلب، حمص، حماة، اللاذقية، طرطوس، إدلب، دير الزور، الرقة، الحسكة، درعا، السويداء، القنيطرة

## 🛠️ التقنيات

Next.js 14, TypeScript, Tailwind CSS, Prisma, PostgreSQL, Zustand, Framer Motion

## 🌐 النشر

```bash
npm install
npx next build
npx next start
```

أو انشر على Vercel بنقرة واحدة.
